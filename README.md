# SyntaxWear - E-commerce de Tênis & Sneakers

![SyntaxWear Logo](./Images/logo/Logo.png)

A **SyntaxWear** é uma landing page moderna e responsiva desenvolvida para um e-commerce especializado em calçados, tênis e sneakers de alta qualidade. O projeto foca em uma experiência de usuário limpa, destacando produtos através de grids visuais e navegação intuitiva.

## 🚀 Funcionalidades

- **Design Responsivo:** Adaptado para dispositivos móveis, tablets e desktops.
- **Menu de Navegação:** Header fixo com suporte a menu hambúrguer para dispositivos móveis.
- **Seção Hero:** Destaque principal com chamadas para ação (CTAs) claras.
- **Categorias de Produtos:** Navegação rápida por estilos (Casual, Esporte, Moderno, Futurista).
- **Grid de Destaques:** Layout dinâmico utilizando CSS Grid para exibir os modelos principais (ex: Krypton One).
- **Rodapé Completo:** Inclui formulário de newsletter, links sociais e organização de categorias.

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Estrutura semântica para melhor acessibilidade e SEO.
- **CSS3:** 
    - Flexbox e CSS Grid para layouts complexos.
    - Variáveis CSS para padronização de cores e fontes.
    - Media Queries para responsividade.
- **Google Fonts:** Utilização da fonte 'Ubuntu' para uma tipografia moderna.
- **SVG:** Ícones leves e escaláveis para melhor performance.

## 💻 Como Executar o Projeto

1. Clone este repositório ou baixe os arquivos.
2. Certifique-se de que a estrutura de pastas `css/` e `Images/` está correba em relação ao `index.html`.
3. Abra o arquivo `index.html` em qualquer navegador moderno (Chrome, Firefox, Edge, etc.).
